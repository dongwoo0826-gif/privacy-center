# Privacy Center — 개인정보보호 실무 플랫폼

> 개인정보 영향평가 255건, 현장점검 15개 시스템, ISMS-P 인증 대응 등 실무 경험을 바탕으로 구축한 **개인정보보호 업무 도구 포트폴리오**입니다.

개인정보 영향평가·안전성 확보조치·흐름표 작성·동의서 생성 등 개인정보보호 실무에 필요한 가이드와 도구를 **웹 기반 인터랙티브 플랫폼** 형태로 정리했습니다. 단순한 이력 나열이 아니라, 실무에서 반복적으로 수행해 온 업무를 **직접 활용 가능한 도구**로 구현하여 역량을 증명하고자 했습니다.

<br/>

## 소개

**이동우** · 개인정보보호 실무자

민간 대기업 환경에서 개인정보 영향평가, 개인정보처리시스템 운영·현장점검, ISO·ISMS-P 인증심사 대응을 수행하며, 보호조치 이행 점검과 보안 통제 기준 개선 업무를 담당하고 있습니다.

'문서상 존재하는 통제'가 아닌 **'실제 운영 환경에서 작동하는 통제'** 를 기준으로 일하며, 법적 요구사항을 현업의 언어로 풀어내어 보안이 업무 과정 안에서 자연스럽게 작동할 수 있도록 만드는 것을 목표로 합니다.

<br/>

## 직무 역량

### 개인정보 영향평가
- 신규 도입·변경·종료를 포함한 **총 255건**의 영향평가 수행 (완료 210건+)
- 개인정보 수집부터 파기까지 전 생명주기 처리 흐름 분석 및 법령 적정성 검토
- 데이터 민감도·침해 영향 범위 기반 리스크 분류 및 단계적 이행 계획 수립

### 운영·현장점검 및 인증 대응
- 데이터 통합·분석 플랫폼 **15개 시스템 현장점검** — 접근통제·암호화·접속기록 관리 등 운영 적정성 검증
- ISO·ISMS-P 인증심사 대응, 위치정보·가명정보·고유식별정보 실태조사 대응
- 개인정보 흐름도·업무별 처리 흐름도 작성 및 인증 증빙자료 정비

### 점검 체계 고도화 및 업무 개선
- AI·CI(연계정보)·암호키·PoC 환경 등 **신규 점검 기준 신설·보완**
- 체크리스트 **75→38개 경량화** 및 최신 법령 기준 반영 (KPI A등급 달성)
- 업무 매뉴얼·FAQ 전면 개정, 반복 엑셀 업무 쉘 스크립트 자동화

### 법령·규제 대응
- **개인정보보호법** 전 조문 기반 실무 적용 (수집·이용·제공·파기·권리보장)
- **안전성 확보조치 기준 고시** (2025.11 개정) 기반 기술적·관리적·물리적 보호조치
- **인공지능기본법** (2026 시행) 반영 AI 서비스 점검 체계 수립
- 정보통신망법 기반 연계정보(CI) 규제 대응

## 기술 스택

이 포트폴리오는 별도의 빌드 도구나 프레임워크 없이 **순수 HTML·CSS·JavaScript**로 구현되었습니다.

- **HTML5** — 시맨틱 구조, `<template>` 태그 활용
- **CSS3** — CSS 변수 기반 디자인 시스템, 반응형 레이아웃
- **Vanilla JS** — SPA 방식 페이지 전환, 동적 데이터 렌더링, 검색·필터링
